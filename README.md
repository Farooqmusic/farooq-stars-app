# Farooq Stars — Mobile App (iOS + Android)

Flutter app for [farooqstars.com](https://www.farooqstars.com) — Western & Vedic
astrology, zodiac signs and compatibility, for curiosity and fun.

- Bundle ID: `com.farooqstars.app`
- Backend: Supabase (same project as the website)
- Builds: Codemagic (`codemagic.yaml`) — iOS -> TestFlight, Android -> .aab
- Languages: English, اردو, हिंदी, العربية (RTL supported)
